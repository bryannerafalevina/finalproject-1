Linux

Similar to Windows, Linux is an open source operating system that can be used freely (free) to be distributed under the GNU license. The first Linux was created by a Finnish student named Linux Torvald. Linux is a derivative of Unix and can work on a variety of computer hardware ranging from inter x86 to RISC.

Mac

The Macintosh Operating System was first created by Apple Inc. with an elegant design and easy to understand by users or brainware. The Macintosh Operating System was the first operating system to use a Graphical User Interface, an operating system that used images (not just text).

Windows

An operating system released by the American company Microsoft. Meanwhile, the operating system is the basic system for operating an electronic device. So, the Windows operating system is an operating system used in an electronic device to which a new program can be added according to the user's wishes, for example in the Windows operating system the Microsoft Office program is provided, to help with typing on the computer.
